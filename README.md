# Times India News Template

A fully customized blogger news website template designed like Times of India. This professional template provides all the essential features needed for a modern news website.

## Features

### Design & Layout
- **Professional News Layout**: Similar to Times of India's design structure
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern CSS Grid & Flexbox**: Clean, organized layouts
- **Typography**: Beautiful font combinations (Roboto + Playfair Display)
- **Color Scheme**: Professional red, gold, and dark color palette

### Header & Navigation
- **Sticky Header**: Always visible navigation while scrolling
- **Top Bar**: Date/time display and social media links
- **Search Functionality**: Built-in search box with JavaScript
- **Multi-level Navigation**: Dropdown menus for categories
- **Mobile Menu**: Hamburger menu for mobile devices

### Content Features
- **Breaking News Ticker**: Animated marquee for urgent updates
- **Featured Article Section**: Prominent display for top stories
- **News Categories**: Politics, Sports, Business, Entertainment, Technology, Health, World
- **Trending News Widget**: Numbered list of popular articles
- **News Grid Layout**: Card-based design for article listings
- **Article Pages**: Full post layouts with sharing options

### Sidebar Widgets
- **Popular Posts**: Most read articles display
- **Categories**: Tag-based content organization
- **Archive**: Monthly/yearly post archives
- **Advertisement Spaces**: Dedicated ad placements

### SEO & Performance
- **Meta Tags**: Complete Open Graph and Twitter Card support
- **Schema Markup**: Structured data for search engines
- **Fast Loading**: Optimized CSS and JavaScript
- **DNS Prefetching**: Improved resource loading
- **Canonical URLs**: Proper URL structure

### Interactive Elements
- **Social Sharing**: Facebook, Twitter, WhatsApp integration
- **Comments System**: Built-in Blogger commenting
- **Search Function**: Real-time search capabilities
- **Back to Top**: Smooth scroll to top button
- **Hover Effects**: Modern CSS transitions

## Installation Instructions

### For Blogger
1. Download the `times-india-news-template.xml` file
2. Go to your Blogger Dashboard
3. Navigate to Theme → Backup/Restore
4. Click "Choose File" and select the downloaded XML file
5. Click "Upload" to apply the template
6. Customize colors and settings as needed

### Customization Options
- Logo: Replace "Times India News" with your website name
- Colors: Modify CSS variables in the template
- Categories: Update navigation menu items
- Social Links: Add your social media URLs
- Advertisement: Replace ad placeholders with real ads
